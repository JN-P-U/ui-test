# Role: Expert Full-stack Developer (Spring Boot)

# Context:
- 환경: 폐쇄망 내부 개발 환경 (인터넷 불가)
- 기술 스택: Spring Boot, HTML5, JavaScript (LocalStorage), CSS
- 제약 사항: 외부 라이브러리(CDN) 사용 불가, 순수 JS와 로컬 리소스로만 구현
- 목적: 프로젝트 수행 중 자주 사용하는 공유폴더 경로, Wiki URL, 가이드 페이지 링크를 관리하는 '개인용 업무 포털' 구축

# Requirements:
1. UI/UX:
   - 상단: 검색바 (제목 또는 경로로 실시간 필터링, 한국어 초성 검색 기능 포함).
   - 중단: 입력 폼 (카테고리, 제목, 경로/URL, 메모).
   - 하단: 저장된 목록 (HTML Table 또는 Card 형태).

2. 핵심 기능:
   - 데이터 저장: 모든 데이터는 Browser LocalStorage에 JSON 형태로 저장 (DB 연결 불필요).
   - 복사 버튼: 각 항목 옆에 [Copy] 버튼을 배치하여 클릭 시 해당 경로를 클립보드에 복사 (navigator.clipboard API 활용).
   - 파일 파싱: 특정 형식의 텍스트 파일(예: 제목|경로|메모)을 붙여넣으면 한꺼번에 LocalStorage에 Import하는 기능.
   - 탐색기 연동 가이드: 'file://' 프로토콜을 사용하거나, 경로 복사 후 윈도우 실행창(Win+R)에 붙여넣기 용이하도록 텍스트 정제.
   - 백업을 위해 신규 저장 데이터 생성 시 .txt 파일로 생성 지정하는 폴더에 page-link_{날짜}.txt 파일을 만들어주고 해당 파일을 읽어서 로드 하는 기능 생성
   - 버튼을 추가하여 백업 기능을 수작업으로 수행 할 수 있는 기능 생성
   - 저장된 링크 목록에 복사 말고 새 탭으로 열기 버튼 추가
   - 북마크바처럼 폴더 구조로 세팅할 수 있도록 기능 수정
   - 폴더는 최 상위 카테고리 하위 서브 폴더 4레벨까지 관리 가능
   - 폴더를 다른 폴더의 하위 또는 루트 폴더로 수정할 수 있고 이때, 하위 폴더 및 포함 파일들도 같이 이동
   
3. 코드 상세:
   - 외부 의존성 없이 단일 HTML 파일 또는 간단한 Spring Boot 컨트롤러/JSP 구조로 작성해줘.
   - 디자인은 가독성이 높도록 다크 모드 지원 또는 차분한 오피스 룩으로 구성해줘.
