# 업무 링크 관리자 (Page Link Manager)

금융권 프로젝트 수행 중 자주 쓰는 **공유폴더 경로 · Wiki URL · 가이드 링크**를 한 곳에서 관리하는 개인용 업무 포털입니다.

- 외부 라이브러리 없음 — 단일 HTML 파일, 브라우저에서 바로 실행
- 모든 데이터는 **브라우저 LocalStorage**에 저장 (서버·DB 불필요)
- 다크모드 지원

---

## 실행 방법

```
page-link-manager.html 파일을 브라우저에서 열기
```

> 폐쇄망 환경에서도 인터넷 연결 없이 동작합니다.

## 주요 기능

### 링크 추가·수정·삭제

| 필드 | 설명 |
|------|------|
| 카테고리 | `Wiki`, `공유폴더`, `가이드` 등 자유 입력. 기존 카테고리는 자동완성 제공 |
| 제목 | 링크를 식별할 이름 |
| 경로/URL | UNC 경로(`\\server\share`) 또는 URL(`https://...`) |
| 메모 | 간단한 설명 (선택) |
| 태그 | 공백으로 구분. 검색 시 활용 |

- 목록의 ✏️ 버튼 클릭 → 폼에 내용 불러와 수정 가능
- 목록의 🗑 버튼 클릭 → 삭제 (확인 팝업)

### 경로 복사

각 행의 **[복사]** 버튼을 클릭하면 해당 경로가 클립보드에 복사됩니다.

- `file:///` 프리픽스는 자동 제거하여 `\\` 형태로 정제
- Win+R (실행창)에 바로 붙여넣기 가능

### 검색 (한국어 초성 지원)

검색창에 입력하면 제목 · 경로 · 메모 · 태그를 실시간으로 필터링합니다.

```
예) "ㄱ"    → "가이드", "공유폴더" 등 ㄱ으로 시작하는 항목 검색
    "공유"  → "공유폴더" 포함 항목 검색
    "배포"  → 제목·메모·태그에 "배포"가 포함된 항목 검색
```

### 가져오기 (Import)

헤더의 **[📥 가져오기]** 버튼을 클릭하면 텍스트를 붙여넣어 일괄 등록할 수 있습니다.

**형식** (한 줄에 항목 하나):
```
카테고리|제목|경로/URL|메모
```

**예시**:
```
Wiki|개발 위키|https://wiki.internal/dev|팀 개발 표준 문서
공유폴더|배포 스크립트|\\192.168.1.10\deploy\scripts|운영 배포 시 사용
가이드|코딩 컨벤션|https://wiki.internal/guide|사내 코딩 규칙
```

- 구분자: `|` (파이프) 또는 `Tab` 모두 지원
- 메모·태그는 생략 가능 (3컬럼도 허용)
- `#`으로 시작하는 줄은 주석으로 처리되어 건너뜀

### 내보내기 (Export)

헤더의 **[📤 내보내기]** 버튼을 클릭하면 현재 저장된 전체 목록을 `links_YYYY-MM-DD.txt` 파일로 다운로드합니다. 가져오기 형식과 동일하므로 다른 PC에서 그대로 불러올 수 있습니다.

### 정렬

테이블 헤더(카테고리 · 제목 · 경로/URL · 등록일)를 클릭하면 오름차순/내림차순 정렬됩니다.

---

## 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl + F` | 검색창 포커스 |
| `Ctrl + Enter` | 폼 입력 중 저장 |
| `Esc` | 가져오기 모달 닫기 / 폼 초기화 |

---

## 데이터 저장 위치

브라우저 LocalStorage (`plm_links_v2` 키)에 JSON 형식으로 저장됩니다.

- 같은 브라우저·같은 PC에서는 새로고침 후에도 데이터 유지
- 다른 PC로 이전할 때는 **내보내기 → 가져오기** 활용

---

## 경로 유형 자동 인식

| 경로 패턴 | 아이콘 | 유형 |
|-----------|--------|------|
| `\\server\share`, `C:\path\` | 📁 | 파일 경로 |
| `file:///...` | 📁 | 파일 경로 |
| `http://`, `https://` 등 | 🌐 | URL / Wiki |
