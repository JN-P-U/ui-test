1. 파읾명
- MAL_{서비스코드}_AC02(단위테스트케이스결과서)_UT_[화면ID]_[화면명]_V1.0.xlsx

2. 시트
- 겉표지
- 개정이력
- 단위테스트케이스
  - 셀 : 입력 항목
    => [] : 사용자 키인
    => {} : 프로그램이 입력
    => 텍스트 : 하드코딩
    - A1 : 업무분류
    - B1 : [입력]
    - C1 : Level1
    - D1 : [입력]
    - E1 : Level2
    - F1 : [입력]
    - G1 : Level3
    - H1 : [입력]
    - I1 : Level4
    - J1 : [입력]
    - K1 : 작성자
    - L1 : [입력]
    - A2 : 단위테스트ID
    - B2 - D2 : {UT_프로그램ID}
    - E2 : 단위테스트명
    - F2 - J2 : [입력]
    - K2 : 작성일
    - L2 : YYYY-MM-DD
    - A3 - I3 : 단위테스트 케이스
    - J3 - L3 : 단위테스트 결과
- 공통체크리스트
- 단위테스트케이스결과증빙
  - 단위테스트 결과 증빙 : A1 - I1
  - 케이스번호 : A2
  - 테스트 결과 증빙 : B2 - I2
  - 번호 : A3, A4...
  - 증빙 : B3, B4...
